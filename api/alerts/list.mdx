---
title: "List Alerts"
api: "GET /api/alerts"
description: "Retrieve all security alerts with optional filtering"
---

## Authentication

Requires a valid session token in the `Authorization` header:

```
Authorization: Bearer <session_token>
```

## Query Parameters

<ParamField query="type" type="string" optional>
  Filter alerts by type. Valid values:
  - `prompt_drift`
  - `prompt_size`
  - `budget_warning`
  - `budget_exceeded`
  - `onchain_denied`
  - `rate_limit_hit`
  - `anomalous_volume`
  - `new_endpoint`
  - `time_anomaly`
  - `high_severity_denied_action`
</ParamField>

<ParamField query="dismissed" type="boolean" optional>
  Filter alerts by dismissal status. Set to `true` to show only dismissed alerts, `false` for active alerts only.
</ParamField>

<ParamField query="limit" type="integer" optional>
  Maximum number of alerts to return
</ParamField>

<ParamField query="skip" type="integer" optional>
  Number of alerts to skip for pagination (default: 0)
</ParamField>

## Response

<ResponseField name="alerts" type="array">
  Array of alert objects
  
  <Expandable title="Alert object">
    <ResponseField name="id" type="string">
      Alert identifier (format: `alert_<number>`)
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Alert type (see query parameter options above)
    </ResponseField>
    
    <ResponseField name="severity" type="string">
      Alert severity level: `critical` or `warning`
    </ResponseField>
    
    <ResponseField name="service" type="string">
      Service that triggered the alert (e.g., `openai`, `binance`)
    </ResponseField>
    
    <ResponseField name="message" type="string">
      Human-readable alert message
    </ResponseField>
    
    <ResponseField name="timestamp" type="integer">
      Unix timestamp when the alert was created
    </ResponseField>
    
    <ResponseField name="dismissed" type="boolean">
      Whether the alert has been dismissed
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Responses

<ResponseField name="error" type="string">
  Error message if the request fails
</ResponseField>

- **401 Unauthorized**: Missing or invalid session token
- **500 Internal Server Error**: Database error

<RequestExample>

```bash cURL
curl -X GET 'https://localhost:3100/api/alerts?type=prompt_drift&dismissed=false' \
  -H 'Authorization: Bearer fn_sess_abc123def456'
```

```python Python
import requests

headers = {
    "Authorization": "Bearer fn_sess_abc123def456"
}

params = {
    "type": "prompt_drift",
    "dismissed": False,
    "limit": 50
}

response = requests.get(
    "https://localhost:3100/api/alerts",
    headers=headers,
    params=params
)

print(response.json())
```

```javascript JavaScript
const response = await fetch(
  'https://localhost:3100/api/alerts?type=prompt_drift&dismissed=false',

    }
  }

```
