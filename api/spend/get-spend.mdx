---
title: "Get Spend Data"
api: "GET /api/spend"
description: "Retrieve spend tracking data and budget information"
---

## Authentication

Requires a valid session token in the `Authorization` header:

```
Authorization: Bearer <session_token>
```

## Query Parameters

<ParamField query="days" type="integer" optional>
  Number of days of spend history to retrieve. Defaults to `spend_history_days` from config. Maximum value is capped by the config setting.
</ParamField>

## Response

<ResponseField name="enabled" type="boolean">
  Whether spend tracking is enabled (`true`) or disabled (`false`)
</ResponseField>

<ResponseField name="config" type="object">
  Spend tracking configuration settings (only present if enabled)
  
  <Expandable title="Config object">
    <ResponseField name="track_spend" type="boolean">
      Whether spend tracking is enabled
    </ResponseField>
    
    <ResponseField name="spend_history_days" type="integer">
      Maximum number of days of history stored
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="daily" type="array">
  Array of daily spend records aggregated by service and date (only present if enabled)
  
  <Expandable title="Daily spend record">
    <ResponseField name="service" type="string">
      Service name (e.g., `openai`, `anthropic`, `binance`)
    </ResponseField>
    
    <ResponseField name="date" type="string">
      Date in `YYYY-MM-DD` format
    </ResponseField>
    
    <ResponseField name="cost_usd" type="number">
      Total cost in USD for this service on this date
    </ResponseField>
    
    <ResponseField name="amount" type="number">
      Alias for `cost_usd` (same value)
    </ResponseField>
    
    <ResponseField name="request_count" type="integer">
      Number of requests made to this service on this date
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="budgets" type="object">
  Budget configuration and current spend by service (only present if enabled). Object keys are service names.
  
  <Expandable title="Budget entry">
    <ResponseField name="daily_limit" type="number">
      Daily budget limit in USD
    </ResponseField>
    
    <ResponseField name="spent_today" type="number">
      Amount spent today in USD
    </ResponseField>
    
    <ResponseField name="warning_pct" type="integer">
      Warning threshold percentage (from config)
    </ResponseField>
    
    <ResponseField name="warning_active" type="boolean">
      Whether the warning threshold has been reached today
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Responses

- **401 Unauthorized**: Missing or invalid session token
- **500 Internal Server Error**: Database error

<Info>
  If spend tracking is disabled in the configuration (`track_spend: false`), the response will only contain `{"enabled": false}`.
</Info>

<RequestExample>

```bash cURL
curl -X GET 'https://localhost:3100/api/spend?days=7' \
  -H 'Authorization: Bearer fn_sess_abc123def456'
```

```python Python
import requests

headers = {
    "Authorization": "Bearer fn_sess_abc123def456"
}

params = {
    "days": 7
}

response = requests.get(
    "https://localhost:3100/api/spend",
    headers=headers,
    params=params
)

data = response.json()
print(f"Enabled: {data['enabled']}")

if data['enabled']:
    for record in data['daily']:
        print(f"{record['date']}: {record['service']} spent ${record['cost_usd']:.2f}")
```

```javascript JavaScript
const response = await fetch(
  'https://localhost:3100/api/spend?days=7',
  {
    headers: {
      'Authorization': 'Bearer fn_sess_abc123def456'
    }
  }
);

const data = await response.json();
console.log('Enabled:', data.enabled);

if (data.enabled) {
  data.daily.forEach(record => {
    console.log(`${record.date}: ${record.service} spent $${record.cost_usd.toFixed(2)}`);
  });
}
```

</RequestExample>
